🔧 Supabase Setup

Quick Setup (5 minutes):
  1. Go to Supabase
  2. Create a new project (any name)
  3. Go to Settings → API
  4. Copy your Project URL and anon public key
  5. Create a table called lagers with columns:
    • id (int8, primary key, auto-increment)
    • name (text)
    • country (text)
    • your_rating (int2)
    • friend1_rating (int2)
    • friend2_rating (int2)
    • friend3_rating (int2)
    • friend4_rating (int2)
    • friend5_rating (int2)
    • friend6_rating (int2)
    • created_at (timestamptz, default: now())
  6. Enable RLS and add a policy to allow all operations (for testing)
